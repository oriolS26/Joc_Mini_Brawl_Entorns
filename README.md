# 🥊 Roblox Mini Brawl

Un microvideojoc de lluita estil **Sumo** en un entorn 2.5D desenvolupat a Roblox Studio.

## 📝 Descripció
Roblox Mini Brawl és un prototip centrat en les mecàniques físiques. El jugador ha d'empènyer el seu oponent fora d'una plataforma flotant. El moviment està restringit a un pla bidimensional per simplificar la jugabilitat i centrar-se en el *timing* dels impactes.

## 🚀 Característiques
* **Perspectiva Fixa:** Càmera scriptada que no canvia l'angle.
* **Física Pura:** Els atacs fan servir el sistema d'impulsos de Roblox.
* **IA de combat:** Un enemic que persegueix el jugador de forma autònoma.
* **Sistema de Vides:** Partides ràpides a 3 vides.

## 🛠️ Eines utilitzades
* **Motor:** [Roblox Studio](https://www.roblox.com/create)
* **Llenguatge:** Luau
* **Sincronització:** [Rojo](https://rojo.space/)
* **Control de versions:** Git & GitHub

## 📦 Instal·lació i Ús
1. Clona el repositori.
2. Obre la terminal i executa `./rojo serve`.
3. Obre Roblox Studio, instal·la el plugin de Rojo i prem **Connect**.
